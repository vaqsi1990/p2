# ტემპლეიტის სურათის დაყენება

## სად შეინახოთ ტემპლეიტის სურათი

ტემპლეიტის სურათი (დაბადების დღის ილუსტრაცია) შეინახეთ შემდეგ ადგილას:

### ვარიანტი 1: Default ტემპლეიტი (რეკომენდებული)

1. დააყენეთ ტემპლეიტის სურათი `assets/img/` ფოლდერში
2. დაარქვით მას სახელი, მაგალითად: `birthday-template.png`
3. გახსენით `upload.js` ფაილი
4. იპოვეთ ეს ხაზი:
   ```javascript
   const DEFAULT_TEMPLATE_PATH = 'http://localhost:3000/assets/img/birthday-template.png';
   ```
5. შეცვალეთ `birthday-template.png` თქვენი ფაილის სახელით

### ვარიანტი 2: ატვირთვა ყოველ ჯერზე

თუ გსურთ ყოველ ჯერზე ატვირთოთ ტემპლეიტი:
1. გახსენით `upload.html` გვერდი
2. დააჭირეთ "ატვირთე ახალი ტემპლეიტი" ღილაკს
3. აირჩიეთ ტემპლეიტის სურათი
4. სურათი ავტომატურად აიტვირთება Supabase Storage-ში

## რეკომენდაცია

რეკომენდებულია ვარიანტი 1 (Default ტემპლეიტი), რადგან:
- სურათი ინახება პროექტში და ყოველთვის ხელმისაწვდომია
- არ საჭიროებს ყოველ ჯერზე ატვირთვას
- უფრო სწრაფია

## ფაილის ფორმატი

ტემპლეიტის სურათი უნდა იყოს:
- PNG, JPG, ან JPEG ფორმატში
- რეკომენდებული ზომა: 1024x1024 ან მეტი
- მაქსიმალური ზომა: 10MB
